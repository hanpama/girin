## Development

### Initialization

```sh
# 1. clone the repository
git clone https://github.com/hanpama/girin

# 2. cd into it
cd girin

# 3. install dependency and link packages
npm run bootstrap

# 4. run typescript compiler server
npm run watch
```

### Testing and Debugging

```
npm test
```
